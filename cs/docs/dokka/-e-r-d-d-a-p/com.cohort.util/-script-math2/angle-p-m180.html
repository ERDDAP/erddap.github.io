anglePM180

[JVM]
fun anglePM180(degrees: Double): Double

This converts an angle (in degrees) into the range >=-180 to <180 (180 becomes -180).

Return​

the angle (in degrees) in the range >=-180 to <180. If isMV(angle), it returns 0.

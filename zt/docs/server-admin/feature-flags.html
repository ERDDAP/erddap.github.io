特性旗號

此頁面記錄系統中可用的設定旗 。 這些旗子控制著不同的特性、實驗能力以及遺傳行為。

圖例​

  • 穩定 : 預期為長期旗號, 讓管理員改變功能 。 安全生产。
  • 測試 : 特徵已經準備好了 或被定在目標值,
  • 在建: 目前硬碼在編碼中是錯的, 不管設定如何 。 此功能尚未準備好使用 。

Tomcatのインストールに問題がありますか?​

  • Linux と Mac では、Tomcat に到達できない場合や ERDDAP™ (またはおそらく、ファイアウォールの外でコンピュータからそれらに到達することはできません) , Tomcat が 8080 ポートを listen しているかどうかを試すことができます。 (ルートとして) サーバのコマンドラインで:

    netstat -tuplen | grep 8080

これは、次のようなもので1行を返すべきです。

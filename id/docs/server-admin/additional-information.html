  • Tentukan format file untuk file data yang akan dibuat. Untuk dataset gridded, untuk NCEI, gunakan.ncSitemap Untuk set data tabel, untuk NCEI, gunakan.ncLoginjika itu adalah pilihan; jika tidak menggunakan.ncSitemap
  • Tentukan jenis file pencernaan untuk dibuat untuk setiap file data dan untuk seluruh paket arsip: MD5, SHA-1, atau SHA-256. pencernaan file menyediakan cara untuk klien (Sitemap) untuk menguji apakah file data telah menjadi rusak.

Sunny Hospital skrev en flott blogg om å få tilgang til private datasett programmatisk.

Eksemplet er i Python .. Du kan tilpasse det til andre språk. Hvis du gjør det, ville det være bra hvis du kunne dele arbeidsflyten din for å gjøre det lettere for andre.

Vær så snill [Les den på bloggen](https://shospital.github.io/blog/posts/blog-post/erddap_private_dataset.html) ..

Merk fra ERDDAP Utviklere for fremtiden. Forbedre prosessen for å få tilgang til private datasett er programmert på listen eller [Prioritetsprosjekter](https://github.com/ERDDAP/erddap/issues/158) ..

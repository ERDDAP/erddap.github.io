Sunny Hospital ay sumulat ng isang malaking blog post tungkol sa pag-access ng mga pribadong datasets programmatly.

Ang halimbawa ay nasa Python . Maaari mo itong ibagay sa ibang wika. Kung gagawin mo ito, makabubuti kung makakasama mo ang iyong trabaho upang maging mas madali ito para sa iba.

Pakisuyo [basahin ito sa blog](https://shospital.github.io/blog/posts/blog-post/erddap_private_dataset.html) .

Pansinin mula sa ERDDAP mga developer para sa hinaharap. Ang pagpapabuti ng proseso ng pag-akses ng mga pribadong datos ng programmatally ay nasa listahan o sa [Mga Proyekto Bago ang Kasal](https://github.com/ERDDAP/erddap/issues/158) .

Sunny Hospital skrev ett bra blogginlägg om att få tillgång till privata dataset programmatiskt.

Exemplet är i Python . Du kanske kan anpassa den till andra språk. Om du gör det skulle det vara bra om du kan dela ditt arbetsflöde för att göra det lättare för andra.

Vänligen vänligen [Läs den på bloggen](https://shospital.github.io/blog/posts/blog-post/erddap_private_dataset.html) .

Notera från ERDDAP utvecklare för framtiden. Förbättra processen för att få tillgång till privata dataset programmatiskt finns på listan eller [Prioriterade projekt](https://github.com/ERDDAP/erddap/issues/158) .

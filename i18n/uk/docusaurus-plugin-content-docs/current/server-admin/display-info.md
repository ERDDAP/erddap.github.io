#  `дисплей Новини` і `дисплейАтрибут` Панчохи

## Опис
Ця функція дозволяє відображати глобальні атрибути вашого вибору на сторінці даних `Інформація` ряд.

## Інструкція з використання
Ці теги можна використовувати тільки з `Сакс парсер` й Щоб увімкнути і використовувати їх, слідуйте за цими кроками:

1.  **Увімкніть SAX Parser** :
Додайте наступний рядок до вашого `Налаштування.xml` Файл:
   ```xml
   <useSaxParser>true</useSaxParser>
   ```

2.  **Додати товари ` datasets.xml ` ** :
У ` datasets.xml ` файл, включає два теги верхнього рівня:
   ```xml
   <displayInfo></displayInfo>
   <displayAttribute></displayAttribute>
   ```

3.  **За замовчуванням Behavior** :
   - Якщо ці теги не додаються або зліва порожній у ` datasets.xml ` Файл, значення за замовчуванням застосовуються в наступному:
     -  `дисплей Новини` : `Резюме,Ліцензія` 
     -  `дисплейАтрибут` : `резюме,ліцензія` 

4.  **Забезпечення консистенції** :
Кількість кома-сепарованих значень в обох `дисплей Новини` і `дисплейАтрибут` теги повинні бути однаковими.

## Як працює
- Про нас `дисплейАтрибут` Теги визначає глобальні атрибути (визначається в межах&lt; ` addAttributes ` &gt; теги) для кожного набору даних.
- Відповідні значення в `дисплей Новини` теги відображаються як етикетки в `Інформація` ряд УІ.
- Після того, як користувач захопить показаними мітками, з'явиться інструмент, що показує значення глобального атрибуту.

## Приклад
```xml
<displayInfo>Display1,Display2</displayInfo>
<displayAttribute>att1,att2</displayAttribute>
```

### Приклад глобальних атрибутів даних:
```xml
<att name="att1">This is att1</att>
<att name="att2">This is att2</att>
```

### UI Бігавіор:
- слова `Дисплей1` і `Показати2` буде відображатися в `Інформація` ряд на УІ.
- При похилому інструментарію будуть відображати відповідні значення атрибутів:
  -  `Дисплей1` Завантажити Aptip для Windows 7, 8, 10
  -  `Показати2` Завантажити Tooltip для Windows 7, 8, 10

## Нотатки
- Забезпечити імена атрибутів, вказані в `дисплейАтрибут` тег відповідає глобальним атрибутам, визначеним у даних.
- Невірно або відсутні атрибути будуть вводити повідомлення про помилки.

За цими кроками можна налаштувати налаштування `Інформація` ряд на сторінці даних для відображення відповідних глобальних атрибутів з відповідними інструментами.

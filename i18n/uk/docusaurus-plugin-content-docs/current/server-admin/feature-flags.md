# Функція Прапори

Ця сторінка містить прапори конфігурації, доступні в системі. Ці прапори контролюють різні функції, експериментальні можливості та поведінки спадщини.

##  **Прапорний цикл Легенда** 

*  **Стабільний:** Увімкніть як довгострокові прапори, щоб дозволити адміністраторам змінити функціонал. Сейф для виробництва.
*  **Тестування:** Особливості, які готові до тестування. Це буде або випускник "Стабільний" або в кінцевому підсумку буде встановлена до їх цільової цінності і має прапор вилучений.
*  **Під будівництвом:** В даний час жорсткийкодований помилково в коді, незалежно від конфігурації. Функція ще не готова до використання.

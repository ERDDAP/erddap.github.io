#  `ڈسپلے معلومات` اور `طےشدہ` تاج

## تفصیل
یہ خصوصیت آپ کو ڈیٹا سیٹز کے صفحہ پر اپنے انتخاب کی عالمی خصوصیات کو ظاہر کرنے کے قابل بناتی ہے۔ `معلومات` قطار.

## ہدایات کا استعمال
ان ٹیگوں کو صرف ساتھ استعمال کیا جا سکتا ہے۔ `سبکدوش` . . انہیں قابل بنانے اور استعمال کرنے کے لیے، ان اقدامات پر عمل کریں:

1.  **SAX Parser کو فعال کرتا ہے۔** :
نیچے دی گئی لائن کو شامل کریں `تیار` فائل:
   ```xml
   <useSaxParser>true</useSaxParser>
   ```

2.  **ٹیگ شامل کریں ` datasets.xml ` ** :
میں ` datasets.xml ` فائل میں دو اوپری سطح کے ٹیگ شامل ہیں:
   ```xml
   <displayInfo></displayInfo>
   <displayAttribute></displayAttribute>
   ```

3.  **طے شدہ** :
   - اگر ان ٹیگوں کو ملا یا خالی کر نہ دیا جائے تو ` datasets.xml ` فائل، متناسب قدروں کا اطلاق اس طرح کیا جاتا ہے:
     -  `ڈسپلے معلومات` : `مصر` 
     -  `طےشدہ` : `خلاصہ` 

4.  **غیر متصل** :
دونوں میں حسابی مقداروں کی تعداد `ڈسپلے معلومات` اور `طےشدہ` ٹیگز ایک ہی ہونا ضروری ہے.

## مثال
```xml
<displayInfo>Display1,Display2</displayInfo>
<displayAttribute>att1,att2</displayAttribute>
```

### Dataset Glomals مثال:
```xml
<att name="att1">This is att1</att>
<att name="att2">This is att2</att>
```

### وعدہ :
- الفاظ `طےشدہ` اور `طےشدہ` عنقریب ظاہر کی جائے گی `معلومات` یو .
- جب یہ واضح ہو جائے گا تو آلات کی نقل و حمل سے متعلقہ اقدار ظاہر ہوں گی:
  -  `طےشدہ` :Toltip ظاہر کرتا ہے کہ یہ Att1_ ہے۔
  -  `طےشدہ` :Toltip ظاہر کرتا ہے کہ یہ Att2_ ہے۔

## نوٹ
- متعین کردہ ناموں کو متعین کریں `طےشدہ` اعداد و شمار میں طے شدہ عالمی خصوصیات کا مقابلہ کریں۔
- لاگ انسس یا گمنام خصوصیات میں لاگ ان

اِن طریقوں پر عمل کرنے سے آپ اِس کا جائزہ لے سکتے ہیں `معلومات` اعداد و شمار کے صفحے پر متعلقہ معلوماتی خصوصیات کو ظاہر کرنے کے لئے جن میں مشترکہ آلات کی مدد سے

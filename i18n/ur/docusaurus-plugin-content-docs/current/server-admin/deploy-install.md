# نصب کریں
توڑوں کی تمثیل ERDDAP™ آپ کے سرور پر

 ERDDAP™ کسی بھی سرور پر چل سکتا ہے جو معاون ہو۔ Java اور ٹومکاٹ (اور دوسرے اطلاقی سرورز جیسے جیٹی، لیکن ہم ان کی حمایت نہیں کرتے) . .
 ERDDAP™ لینکس پر ٹیسٹ کیا گیا ہے۔ (ایمزون کی اے وی ایس پر بھی شامل ہے۔) .، میک اور ونڈوز کمپیوٹر.

*  **بیکر** -- ہم فراہم کرتے ہیں [ ERDDAP™ ڈرنے والے برتن میں](https://hub.docker.com/r/erddap/erddap) 
اور IOS اب ایک تجویز دیتا ہے [فوری طور پر راہنمائی کا آغاز کریں ERDDAP™ ایک ڈرگ میں](https://ioos.github.io/erddap-gold-standard/index.html) . .
یہ معیار ہے۔ ERDDAP™ نصب کریں، ایک ڈرافٹ میں.
ڈاکر کے ذریعے باقاعدگی سے ورزش کریں ۔ [بیکر دستاویزات](https://github.com/ERDDAP/erddap/blob/main/DOCKER.md) . .
اگر آپ پہلے سے ڈریک استعمال کریں گے تو آپ ضرور ڈریک ورژن کو ترجیح دیں گے۔
اگر آپ بادل کی خدمات پر دوڑنے کے لئے جا رہے ہیں تو شاید آپ ڈریک ورژن کو ترجیح دیں گے۔
*  **ایمیزون** -- اگر آپ کوئی منصوبہ بندی کر رہے ہیں ERDDAP™ ایک امیزون ویب سروسز EC2 پر یہ دیکھیں [امی‌ابو کی ویب‌سائٹ پر زیادہ توجہ دیں](/docs/server-admin/additional-information#amazon) پہلے.
*  **لینکس اور میکس** -- ERDDAP™ لینکس اور میک کمپیوٹر پر زبردست کام کرتا ہے۔ نیچے دی گئی ہدایات کو دیکھیں ۔
*  **ونڈو** -- ونڈوز امتحان کے لیے ٹھیک ہے۔ ERDDAP™ اور ذاتی استعمال کے لیے (نیچے دی گئی ہدایات کو دیکھیں) :
لیکن ہم اسے عوام کے لئے استعمال نہیں کرتے ERDDAP™ ذمہ‌داری ۔ چلاو ERDDAP™ ونڈوز پر مسائل ہو سکتے ہیں:
نہیں، ERDDAP™ ہو سکتا ہے کہ جلد ہی فائل حذف کرنے اور/یا نام تبدیل کرنے میں ناکام رہے۔ یہ غالباً اینٹی وائرس سافٹ ویئر کی وجہ سے ہے۔
   (مثلاً مکافی اور نورٹن سے۔) جو وائرسوں کے لیے فائلوں کا جائزہ لیتا ہے۔ اگر آپ اس مسئلہ میں چلا جائے تو
(جو غلطی میں پیغامات سے دیکھی جا سکتی ہے۔ [لاگ.txt](/docs/server-admin/additional-information#log) فائل قسم
"Un ناقابل فراموش . .", اینٹیویرس سافٹ ویئر کی ترتیبات کو تبدیل کرنے سے مسئلہ جزوی طور پر حل ہو سکتا ہے۔ یا اس کی بجائے لینکس یا میک سرور استعمال کریں۔

 **معیار ERDDAP™ لینکس، میکس اور ونڈوز کمپیوٹر کے لیے تنصیب کردہ ہدایات یہ ہیں:** 

0. کوئی بھی پلگ ان نصب کریں غیر نیٹو مشینوں پر (لینکس اور میک) ، آپ کو CS کی ضرورت ہے.

### ذکر{#memory} 

سیٹ ٹومکاٹ کا ماحول وریبلز ہے۔

* لینکس اور میکس پر:
فائل بنام `tomcat/bin/Setenv.sh بنائیں‘‘ (یا ریڈ ہیٹ انٹرپرائز لینکس میں \\[ روس \\] ، تدوین‘ ~ٹومکات/conf/tomcat10.conf‘‘) ٹامکاٹ کا ماحول تبدیل کرنے کے لئے.
یہ فائل 'ٹومکاٹ/بین/اسطپ.sh' اور 'Shuutdown' استعمال کی جائے گی. فائل میں کچھ یوں ہونا چاہیے:
  ```

  export JAVA_OPTS='-server -Djava.awt.headless=true -Xmx1500M -Xms1500M'
  export TOMCAT_HOME=/usr/local/apache-tomcat-10.0.23
  export CATALINA_HOME=/usr/local/apache-tomcat-10.0.23
  ```
   (مگر اپنے کمپیوٹر سے ڈائریکٹری نام تبدیل کریں) . .
   (اگر آپ نے پہلے سے مقرر کیا تھا `JRE_Gmail' تو، آپ اسے ہٹا سکتے ہیں.)

### رنگ (مخصوص)  {#proxy} 

 ERDDAP™ اسے معیاری ایچ ٹی پی بندرگاہوں پر خدمات انجام دینے کی اجازت دینے کے لیے ایک ویب سیریز کے پیچھے دوبارہ کیمرا لگانے کی ضرورت ہے (80 اور 443) . .
SSL/TLS sperm کو اکثر ویب سیارچے میں بھی شمار کیا جاتا ہے۔ زکوٰۃ ہر اندراج کے تقاضوں پر منحصر ہے۔

#### اپسکا{#apache} 

1. اس میں ترمیم کی گئی کہ `Mad_prixo' اور `Add_prixi - http `آپ لوڈ شدہ:

```
LoadModule proxy_module modules/mod_proxy.so
LoadModule proxy_http_module modules/mod_proxy_http.so
```

2. پہلے سے تیار <VirtualHost> ‘‘ ٹیگ (اگر ایک ہے) )، یا فائل کے آخر میں ایک جمع کیا جائے:
```
<VirtualHost *:80>
   ServerName YourDomain.org
   ProxyRequests Off
   ProxyPreserveHost On
   ProxyPass /erddap http://localhost:8080/erddap
   ProxyPassReverse /erddap http://localhost:8080/erddap
</VirtualHost>
```

اگر ERDDAP™ اس کے علاوہ ایک سڑک پر خدمات انجام دی جاتی ہیں‘/ارداپ‘، اس کے علاوہ 'ایکس' کو انعام یافتہ-Presid-Preser' بھی قرار دیا جاتا ہے۔
راستے _پہلے: `derdap' چلائیں. یہ سیٹ کسی کیلئے مناسب ہوگا ERDDAP™ خدمت انجام دینے والے
`subpath/derdap':

```
RequestHeader set X-Forwarded-Prefix /subpath
```

3. اس کے بعد دوبارہ اپاچی: `Sr/sr/sbin/pachectl - -kwest‘‘ (لیکن بعض اوقات یہ ایک مختلف ڈائریکٹری میں ہوتا ہے۔) . .
         
#### چین{#nginx} 

ینگینکس کی فائل میں ان چیف مقرر کیے:
```
proxy_set_header Host              $http_host;
proxy_set_header X-Real-IP         $remote_addr;
proxy_set_header REMOTE_ADDR       $remote_addr;
proxy_set_header HTTP_CLIENT_IP    $remote_addr;
proxy_set_header X-Forwarded-For   $proxy_add_x_forwarded_for;
proxy_set_header X-Forwarded-Proto $scheme;
```

اگر ERDDAP™ اس کے علاوہ ایک سڑک پر خدمات انجام دی جاتی ہیں‘/ارداپ‘، اس کے علاوہ 'ایکس' کو انعام یافتہ-Presid-Preser' بھی قرار دیا جاتا ہے۔
راستے _پہلے: `derdap' چلائیں. یہ سیٹ کسی کیلئے مناسب ہوگا ERDDAP™ خدمت انجام دینے والے
`subpath/derdap':

```
proxy_set_header X-Forwarded-Prefix /subpath
```


این جی او حاصل کرنے کے لئے ERDDAP™ درست کے ساتھ کام کریں https )، آپ کو ٹومکاٹ سرور کے اندر مندرجہ ذیل Snippet کرنے کی ضرورت ہے۔ <Host> ‘‘ بلاک:
```
<Valve className="org.apache.catalina.valves.RemoteIpValve"
  remoteIpHeader="X-Forwarded-For"
  protocolHeader="X-Forwarded-Proto"
  protocolHeaderHttpsValue="https" />
```

جواب Python پیکج 'xarray' مختلف فارمیٹ میں رسائی، ذیلی انفنٹری اور نظریاتی کشش کے لیے بہت مقبول ہو گیا ہے۔ 'xarray' کے ساتھ اچھا کام کرتے ہیں۔ ERDDAP™ ایک بار جب آپ اسے صحیح طریقے سے استعمال کرنا جانتے ہیں ۔ میں سمجھ جائے گا کہ Python پیکج 'اردوپی' ' ( https://github.com/ioos/erddapy ) سے معلومات تک رسائی حاصل کر سکتے ہیں۔ ERDDAP™ سرور دونوں 'گریڈپ' اور 'سٹرڈپ‘ استعمال کرتے ہیں۔ tabledap '، اور 'رداپ' کے لیے اعداد و شمار کو برآمد کر سکتے ہیں۔ لیکن اگر آپ 'xarray' استعمال کر رہے ہیں اور پیکج استعمال کر رہے ہیں تو پھر ایک پیکج کے اندر صرف کام کرنا اچھا ہو سکتا ہے. ذیل میں ایک مثال 'griddap' dataset کے ساتھ ہے۔

میرے پسندیدہ اعداد و شمار میں سے ایک JPL MURv4.1 ایس ٹی ڈاٹا ہے۔ https://coastwatch.pfeg.noaa.gov/erddap/griddap/jplMURSST41.html . . اگر میں 28 جنوری 2026ء کے لیے اعداد و شمار کا ایک ذیلی مجموعہ کرنا چاہتا ہوں، تو لاٹیٹیوز (20,50) اور لمبے لمبے سایوں (-140، -105) . . . . ERDDAP™ اس کے لئے دادی ہو جائے گا https://coastwatch.pfeg.noaa.gov/erddap/griddap/jplMURSST41.nc?analysed_sst[(2026-01-28T09:00:00Z 1: (2026-01-28T09:00Z) [] (۲۰) سوال : (50) [] (-140) سوال : (-105) [] اور یہ اندازہ لگانا معقول ہے کہ 'کسی' میں آپ یہ استعمال کریں گے۔ لیکن اگر ایسا ہے تو آپ غلطی کر رہے ہیں ۔

اس غلطی کی وجہ یہ ہے کہ 'کسررائی' استعمال ہوتی ہے۔ OPeNDAP   ( https://www.opendap.org ) دور رسائی کے لیے اس کے پروٹوکول اور جبکہ ERDDAP™ ایس ایم پر مبنی ہے۔ OPeNDAP کرس اینڈرسن: ERDDAP™ سرور بھی بطور کردار ادا کر سکتا ہے۔ OPeNDAP سرور، دونوں خدمات کے لیے یہ کام کیسے کیا جاتا ہے۔ (مثال کے طور پر دیکھیں https://coastwatch.pfeg.noaa.gov/erddap/griddap/documentation.html#opendapLibraries ) . . کسی بھی ERDDAP صرف datasetID ، ایک طرح کی بات ہے OPeNDAP حضرات اور xarray سے مطابقت رکھیں گے.

اگر ہم کسی مقامی تک رسائی کے اقدامات کے بارے میں سوچیں NetCDF فائل 'xarray' میں ہم مندرجہ ذیل اقدامات کرتے ہیں:

- فائل فہرست حاصل کرنے کے لیے فائل کا بھرپور راستہ کھولیں
- پہلے مرحلے سے متعلقہ معلومات کو دیکھیں
- ڈیٹا کو زیر کرنے کے لیے مختلف " انتخابی" طریقوں میں سے ایک استعمال کریں۔

اوپر موجود فائل کو سب سے پہلے کھول دو پھر ذیلی تقسیم کریں۔ کے ساتھ 'xarray' استعمال کرنے کے لئے ERDDAP™ اعداد و شمار سے اندازہ ہوتا ہے کہ آپ بھی اسی طرح کرتے ہیں، ایک بار آپ کو احساس ہوتا ہے، جیسا کہ وضاحت میں، ERDDAP™ دستاویز، کہ " فائل کی طرف سفر"، بغیر کسی واپسی کے فارمیٹ اور بغیر کسی کوائن کے، MUR dataset کے معاملے میں https://coastwatch.pfeg.noaa.gov/erddap/griddap/jplMURSST41.
 

ایم آر ڈیٹا سیٹ استعمال کرتے ہوئے، پہلے مجھے ان پیکجوں کو درآمد کرنے کی ضرورت ہے جو استعمال ہوں گے:


```python
import matplotlib.pyplot as plt
import xarray as xr
```

اس کے بعد، جیسا کہ میں نے ڈیٹا سیٹ کو "نام" اور "فل" کھول دیا۔ (یہ حقیقت نہیں کہ یہ فائلوں کی ایک ذیلی شاخ ہے۔) "xr.open_dataset" استعمال کرتے ہوئے


```python
mur_url = "https://coastwatch.pfeg.noaa.gov/erddap/griddap/jplMURSST41"
ds = xr.open_dataset(mur_url, decode_times=True)


```

#  `οθόνη Πληροφορίες` και `χαρακτηριστικό εμφάνισης` Ετικέτες

## Περιγραφή εμπορευμάτων
Αυτή η λειτουργία σας επιτρέπει να εμφανίσετε καθολικές ιδιότητες της επιλογής σας στη σελίδα συνόλων δεδομένων στη σελίδα `Πληροφορίες` Σειρά.

## Οδηγίες Χρήσης
Αυτές οι ετικέτες μπορούν να χρησιμοποιηθούν μόνο με το `Αναλυτής Sax` . Για να τα ενεργοποιήσετε και να τα χρησιμοποιήσετε, ακολουθήστε τα παρακάτω βήματα:

1.  **Ενεργοποίηση του αναλυτή SAX** :
Προσθήκη της ακόλουθης γραμμής σε σας `setup.xml` αρχείο:
   ```xml
   <useSaxParser>true</useSaxParser>
   ```

2.  **Προσθήκη ετικετών ` datasets.xml ` ** :
Στο ` datasets.xml ` αρχείο, περιλαμβάνει δύο ετικέτες ανώτατου επιπέδου:
   ```xml
   <displayInfo></displayInfo>
   <displayAttribute></displayAttribute>
   ```

3.  **Προκαθορισμένη συμπεριφορά** :
   - Εάν αυτές οι ετικέτες δεν προστίθενται ή αφήνονται κενές στο ` datasets.xml ` αρχείο, οι προκαθορισμένες τιμές εφαρμόζονται ως εξής:
     -  `οθόνη Πληροφορίες` : `Περίληψη,License` 
     -  `χαρακτηριστικό εμφάνισης` : `περίληψη, άδεια` 

4.  **Εξασφάλιση συνέπειας** :
Ο αριθμός των τιμών που χωρίζονται σε κόμματα και στις δύο `οθόνη Πληροφορίες` και `χαρακτηριστικό εμφάνισης` Οι ετικέτες πρέπει να είναι ίδιες.

## Πώς Λειτουργεί
- Η `χαρακτηριστικό εμφάνισης` η ετικέτα καθορίζει τα καθολικά χαρακτηριστικά (καθορίζεται εντός του&lt; ` addAttributes ` &gt; tag) για κάθε σύνολο δεδομένων.
- Οι αντίστοιχες τιμές `οθόνη Πληροφορίες` η ετικέτα εμφανίζεται ως ετικέτα στο `Πληροφορίες` σειρά του UI.
- Όταν ο χρήστης αιωρείται πάνω από τις εμφανιζόμενες ετικέτες, θα εμφανιστεί μια εργαλειοθήκη, δείχνοντας την τιμή του παγκόσμιου χαρακτηριστικού.

## Παράδειγμα
```xml
<displayInfo>Display1,Display2</displayInfo>
<displayAttribute>att1,att2</displayAttribute>
```

### Παράδειγμα καθολικών χαρακτηριστικών σετ δεδομένων:
```xml
<att name="att1">This is att1</att>
<att name="att2">This is att2</att>
```

### Συμπεριφορά UI:
- Οι λέξεις `Εμφάνιση 1` και `Εμφάνιση2` θα εμφανίζονται στο `Πληροφορίες` κωπηλατεί στο UI.
- Όταν αιωρούνται, οι υποδείξεις θα εμφανίζουν τις αντίστοιχες τιμές ιδιοτήτων:
  -  `Εμφάνιση 1` : Tooltip εμφανίζει _Αυτό είναι att1_
  -  `Εμφάνιση2` : Tooltip εμφανίζει _Αυτό είναι att2_

## Σημειώσεις
- Εξασφάλιση των ονομάτων ιδιοτήτων που καθορίζονται στο `χαρακτηριστικό εμφάνισης` ετικέτα ταιριάζει με τις καθορισμένες στο σύνολο δεδομένων καθορισμένες καθολικές ιδιότητες.
- Οι λανθασμένες ή ελλείπουσες ιδιότητες θα καταγράφουν τα μηνύματα σφάλματος.

Ακολουθώντας αυτά τα βήματα, μπορείτε να προσαρμόσετε το `Πληροφορίες` γραμμή στη σελίδα συνόλων δεδομένων για την εμφάνιση σχετικών καθολικών χαρακτηριστικών με αντίστοιχες υποδείξεις.

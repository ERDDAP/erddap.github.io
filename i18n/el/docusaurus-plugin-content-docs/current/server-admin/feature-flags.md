# Σημαίες χαρακτηριστικών

Αυτή η σελίδα τεκμηριώνει τις σημαίες διαμόρφωσης διαθέσιμες στο σύστημα. Αυτές οι σημαίες ελέγχουν διάφορα χαρακτηριστικά, πειραματικές δυνατότητες, και κληρονομικές συμπεριφορές.

##  **Υπόμνημα κύκλου ζωής σημαιών** 

*  **Σταθερό:** Προορίζεται ως μακροπρόθεσμη σημαίες για να επιτρέψει στους διαχειριστές να αλλάξουν τη λειτουργικότητα. Ασφαλής για την παραγωγή.
*  **Δοκιμή:** Χαρακτηριστικά που είναι έτοιμα για δοκιμή. Αυτά είτε θα αποφοιτήσουν στο "Stable" ή τελικά θα οριστούν στην τιμή-στόχο τους και θα αφαιρεθεί η σημαία.
*  **Υπό κατασκευή:** Αυτή τη στιγμή κωδικοποιείται σε ψευδή στον κώδικα, ανεξάρτητα από τη διαμόρφωση. Το χαρακτηριστικό δεν είναι ακόμα έτοιμο για χρήση.

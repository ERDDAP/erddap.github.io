# 特徴 旗

このページでは、システム内で利用可能な設定フラグを記述しています。 これらのフラグは、さまざまな機能、実験的な機能、およびレガシーの動作を制御する。

##  **フラッグ・ライフサイクル・レジェンド** 

*  **安定した:** 管理者が機能を変更できるようにする長期的なフラグとして意図しました。 生産のために安全。
*  **テスト:** テストの準備ができた特徴。 これらは「Stable」または最終的に目標値に設定され、フラグが削除されます。
*  **構造の下:** 現在、設定に関係なく、コード内で false にハードコードされます。 機能はまだ使用できません。

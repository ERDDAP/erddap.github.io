#  `प्रदर्शन जानकारी` और `प्रदर्शन` टैग

## विवरण
यह सुविधा आपको डेटासेट पेज पर अपनी पसंद के वैश्विक विशेषताओं को प्रदर्शित करने की अनुमति देती है। `सूचना` पंक्ति।

## उपयोग निर्देश
इन टैगों का उपयोग केवल साथ किया जा सकता है `Sax parser` । उन्हें सक्षम और उपयोग करने के लिए, इन चरणों का पालन करें:

1.  **SAX पार्सर को सक्षम करें** :
अपने लिए निम्नलिखित लाइन जोड़ें `साइटमैप` फ़ाइल:
   ```xml
   <useSaxParser>true</useSaxParser>
   ```

2.  **टैग जोड़ें ` datasets.xml ` ** :
में ` datasets.xml ` फ़ाइल में दो शीर्ष स्तर के टैग शामिल हैं:
   ```xml
   <displayInfo></displayInfo>
   <displayAttribute></displayAttribute>
   ```

3.  **डिफ़ॉल्ट व्यवहार** :
   - यदि इन टैगों को जोड़ा नहीं जाता है या खाली छोड़ दिया जाता है तो ` datasets.xml ` फ़ाइल, डिफ़ॉल्ट मान निम्नानुसार लागू होते हैं:
     -  `प्रदर्शन जानकारी` : `सारांश, लाइसेंस` 
     -  `प्रदर्शन` : `सारांश, लाइसेंस` 

4.  **स्थिरता सुनिश्चित करना** :
दोनों में अल्पसंख्यक मूल्यों की संख्या `प्रदर्शन जानकारी` और `प्रदर्शन` टैग समान होना चाहिए।

## यह कैसे काम करता है
- The The most of the `प्रदर्शन` टैग वैश्विक विशेषताओं को निर्दिष्ट करता है (के भीतर परिभाषित)&lt; ` addAttributes ` प्रत्येक डेटासेट के लिए टैग प्रदर्शित किया जाएगा।
- में संबंधित मान `प्रदर्शन जानकारी` टैग लेबल के रूप में प्रदर्शित होते हैं `सूचना` यूआई की पंक्ति।
- जब उपयोगकर्ता प्रदर्शित लेबल पर होवर्स करता है, तो एक टूलटिप दिखाई देगा, जो वैश्विक विशेषता का मूल्य दिखाता है।

## उदाहरण
```xml
<displayInfo>Display1,Display2</displayInfo>
<displayAttribute>att1,att2</displayAttribute>
```

### डेटासेट ग्लोबल एट्रिब्यूट्स उदाहरण:
```xml
<att name="att1">This is att1</att>
<att name="att2">This is att2</att>
```

### यूआई व्यवहार:
- शब्द `प्रदर्शन1` और `प्रदर्शन2` में प्रदर्शित किया जाएगा `सूचना` यूआई पर पंक्ति।
- जब होवर किया जाता है, तो टूलटिप संबंधित विशेषता मूल्यों को प्रदर्शित करेगा:
  -  `प्रदर्शन1` : टूलटिप शो
  -  `प्रदर्शन2` : टूलटिप शो

## नोट्स
- विशेषता नाम में निर्दिष्ट सुनिश्चित करें `प्रदर्शन` टैग डेटासेट में परिभाषित वैश्विक विशेषताओं से मेल खाते हैं।
- गलत या लापता गुण त्रुटि संदेश लॉग करेंगे।

इन चरणों का पालन करके, आप अनुकूलित कर सकते हैं `सूचना` संबंधित टूलटिप्स के साथ प्रासंगिक वैश्विक विशेषताओं को प्रदर्शित करने के लिए डेटासेट पृष्ठ पर पंक्ति।

# 特性旗

本页面记录系统可用的配置旗. 这些旗帜控制着各种特征,实验能力,以及遗产行为.

##  **旗帜生命周期图例** 

*  **稳定 :** 意为长期旗帜,允许管理员更改功能. 安全生产。
*  **测试 :** 准备测试的地物。 这些将要么毕业于"稳定",要么最终被设定为目标值并被去掉旗帜.
*  **建设中:** 目前无论配置如何,在代码中被硬编码为虚假. 该特征尚未准备使用.

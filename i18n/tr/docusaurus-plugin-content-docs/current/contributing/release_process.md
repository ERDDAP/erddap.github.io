# ERDDAP™Yayın Süreci
* Görüntü karşılaştırma dosyalarının mevcut olduğundan emin olun (Bu, "mvn doğrulama" anlamına gelebilir, eğer sadece ImageComparison grubuna kısıtlamak istiyorsanız, Jetty testlerini hala çalıştırmayı gerektiren notları azaltır.) 
* Güncelleme bağımlılıkları
```
mvn versions:display-dependency-updates   // (displays updates)
mvn versions:use-latest-versions  // (updates dependencies, though sometimes we don’t want to do all of them)
mvn versions:update-properties // (updates versions in the property block)
```
* Update plugins
```
mvn versions:display-plugin-updates // (displays updates, need to manually update)
```
* Bağımlılık güncelleştirmelerinin tüm büyük konfigürasyonlar için hiçbir şeyi bozmadığından emin olmak için testler (Datasets özellikle parsing, başka herhangi bir önemli ayarlar da olsa) 
```
mvn verify
```
* TranslateMessages kullanın.translate () Gerekirse çevirileri güncellemek için
* EDStatic.java gelişim kurdu Sahte mod, sürüm numarasını değiştirin ve sürüm tarihini belirtin.
* Yapın
```
mvn clean
mvn compile
mvn package
```

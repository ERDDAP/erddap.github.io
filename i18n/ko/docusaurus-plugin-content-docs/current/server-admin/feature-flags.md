# 기능 플래그

이 문서는 시스템에서 설정 플래그를 사용할 수 있습니다. 이 깃발은 각종 특징, 실험적인 기능 및 유산 행동을 통제합니다.

##  **Flag Lifecycle 전설** 

*  **안정:** 관리자가 기능을 변경할 수 있도록 장기 플래그로 의도했습니다. 생산에 대한 안전.
*  **시험:** 테스트 준비가 된 기능. 이들은 "Stable"을 졸업하거나 결국 대상 값으로 설정되고 플래그가 제거됩니다.
*  **건축의 밑에:** 현재 구성에 관계없이 코드에서 false로 하드 코딩. 이 기능은 아직 사용할 준비가 없습니다.

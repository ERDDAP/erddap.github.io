# Особенности флагов

Эта страница документирует флаги конфигурации, доступные в системе. Эти флаги контролируют различные функции, экспериментальные возможности и устаревшее поведение.

##  **Легенда жизненного цикла флага** 

*  **Стабильно:** Предназначены как долгосрочные флаги, позволяющие администраторам изменять функциональность. Безопасно для производства.
*  **Испытания:** Особенности, которые готовы к тестированию. Они либо перейдут в «Стабильный», либо в конечном итоге будут установлены на целевое значение и будут удалены.
*  **В стадии строительства:** В настоящее время жестко закодировано для ложного в коде, независимо от конфигурации. Функция еще не готова к использованию.

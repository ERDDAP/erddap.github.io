# แปรงทาสี

เอกสารหน้าเว็บนี้ จะแสดงธงปรับแต่งที่มีอยู่ในระบบ ธงเหล่านี้ควบคุมคุณสมบัติต่างๆ ความสามารถในการทดลอง และพฤติกรรมมรดก

##  **วงเวียนธง** 

*  **ความจุ:** แนะนําให้ใช้ธงระยะยาว อนุญาตให้แอดมินส์เปลี่ยนความสามารถ ปลอดภัยสําหรับการผลิต
*  **ทดสอบ:** คุณสมบัติที่พร้อมสําหรับการทดสอบ เหล่านี้จะจบการศึกษาเป็น "ตาราง" หรือในที่สุดได้ถูกตั้งค่าให้เป็นค่าเป้าหมายของพวกเขา และมีการเอาธงออก
*  **การ ก่อ สร้าง:** ปัจจุบัน มี การ ใช้ รหัส ผิด ไม่ ว่า จะ มี การ ปรับ ปรุง อย่าง ไร ก็ ตาม. คุณสมบัตินี้ยังไม่พร้อมใช้งาน

特性旗

本页面记录了系统中可用的配置旗. 这些旗帜控制着各种特征,实验能力和遗产行为.

旗帜生命周期图例​

  • 稳定 : 意为长期旗帜,允许管理员更改功能. 安全生产。
  • 测试 : 可以测试的特征 这些会要么毕业到"稳定",要么最终被设定到目标值,并被移除国旗.
  • 建设中: 目前无论配置如何,在代码中硬码为虚假. 该特性尚未准备使用.

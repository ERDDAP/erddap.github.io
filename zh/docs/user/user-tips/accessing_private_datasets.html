accessing_private_datasets

Sunny Hospital wrote a great blog post about accessing private datasets programmatically.

The example is in Python. You may be able to adapt it to other languages. If you do, it'd be great if you could share your workflow to make it easier for others.

Please read it on the blog.

Note from the ERDDAP developers for the future. Improving the process for accessing private datasets programmatically is on the list or Priority Projects.
